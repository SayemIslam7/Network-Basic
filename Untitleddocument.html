    <h3>Network Basics for IT Support Professionals</h3>

    <h4>1. Network Types and Topologies</h4>

    <ul>

        <li><b>Local Area Network (LAN)</b>: Connects devices within a limited area, such as an office.</li>

        <li><b>Wide Area Network (WAN)</b>: Connects multiple LANs over large geographic areas.</li>

        <li><b>Wireless LAN (WLAN)</b>: Uses wireless technology to connect devices within a LAN.</li>

        <li><b>Personal Area Network (PAN)</b>: Connects personal devices within a short range, typically using Bluetooth.</li>

    </ul>

    <h4>Network Topologies:</h4>

    <ul>

        <li><b>Star Topology</b>: All devices connect to a central hub or switch.</li>

        <li><b>Bus Topology</b>: All devices share a single communication line.</li>

        <li><b>Ring Topology</b>: Devices are connected in a circular fashion.</li>

        <li><b>Mesh Topology</b>: Devices are interconnected, providing multiple paths for data.</li>

    </ul>

    <h4>2. Essential Networking Devices</h4>

    <ul>

        <li><b>Router</b>: Connects multiple networks and routes data between them.</li>

        <li><b>Switch</b>: Connects devices within a LAN and uses MAC addresses to forward data.</li>

        <li><b>Access Point (AP)</b>: Provides wireless connectivity within a network.</li>

        <li><b>Modem</b>: Converts digital signals to analog for transmission over telephone lines and vice versa.</li>

        <li><b>Firewall</b>: Monitors and controls incoming and outgoing network traffic based on security rules.</li>

    </ul>

    <h4>3. Common Networking Protocols</h4>

    <ul>

        <li><b>TCP/IP (Transmission Control Protocol/Internet Protocol)</b>: Fundamental suite of protocols for internet communication.</li>

        <li><b>HTTP/HTTPS (HyperText Transfer Protocol/Secure)</b>: Protocols for web communication.</li>

        <li><b>FTP (File Transfer Protocol)</b>: Used for transferring files between computers.</li>

        <li><b>DNS (Domain Name System)</b>: Translates domain names to IP addresses.</li>

        <li><b>DHCP (Dynamic Host Configuration Protocol)</b>: Automatically assigns IP addresses to devices on a network.</li>

    </ul>

    <h4>4. Basic IP Addressing and Subnetting</h4>

    <ul>

        <li><b>IP Address</b>: Unique identifier for a device on a network (e.g., IPv4: 192.168.1.1, IPv6: 2001:0db8::1).</li>

        <li><b>Subnet Mask</b>: Divides an IP address into network and host portions (e.g., 255.255.255.0).</li>

        <li><b>Default Gateway</b>: The device that routes traffic from a local network to external networks.</li>

        <li><b>DNS Server</b>: Translates domain names to IP addresses.</li>

    </ul>

    <h4>5. Network Security Basics</h4>

    <ul>

        <li><b>Firewall</b>: Blocks unauthorized access while permitting outward communication.</li>

        <li><b>Encryption</b>: Protects data by converting it into a secure format during transmission.</li>

        <li><b>VPN (Virtual Private Network)</b>: Creates a secure connection over the internet.</li>

        <li><b>Antivirus Software</b>: Protects devices from malware and viruses.</li>

        <li><b>Intrusion Detection Systems (IDS)</b>: Monitors network traffic for suspicious activity.</li>

    </ul>

    <h4>6. Common Network Troubleshooting Techniques</h4>

    <ul>

        <li><b>Ping</b>: Tests connectivity between devices.</li>

        <li><b>Traceroute</b>: Tracks the path packets take to reach their destination.</li>

        <li><b>IPConfig/IFConfig</b>: Displays network configuration details.</li>

        <li><b>NSLookup</b>: Queries DNS to obtain domain name or IP address information.</li>

        <li><b>Wireshark</b>: Captures and analyzes network traffic.</li>

        <li><b>Check Physical Connections</b>: Ensure cables and devices are properly connected.</li>

        <li><b>Restart Devices</b>: Restart routers, switches, and computers to resolve connectivity issues.</li>

        <li><b>Update Drivers/Firmware</b>: Ensure network devices have the latest updates installed.</li>

    </ul>

    <p>This guide provides a foundational understanding of networking essential for IT support professionals, enabling them to effectively manage and troubleshoot network-related issues.</p>
